Print

Функция print выводит переданные ей данные на стандартный поток вывода (в терминал).

            print('hello, world!')
            print('У меня', 10, 'пальцев на руках')

Математические операторы

            print(42)
            print(42 + 21)
            print(42 - 21)
            print(42 * 21)
            print(42 / 21)

Переменная

— ячейка в оперативной памяти компьютера, у которой есть имя и в которую можно положить значение.

Как коробочка со стикером на ней!

            age = 28
            print('Мне', age, 'лет')
            print('В следующем году мне будет', age + 1, 'лет')

Типы данных

  • int — целые числа (42)
  • float — вещественные (дробные) числа (4.2)
  • str — строки ('42', 'Game Over')
  • bool — логическое значение (True / False)

Условный оператор

Условный оператор выполняет некоторое действие, если указанное условие истинно (равно True).

            if True:
                print('Условие истинно!')

Оператор else

            if False:
                print('Условие истинно')
            else:
                print('Условие ложно')

Цикл while

Цикл while (“пока”) позволяет выполнить одну и ту же последовательность действий, пока проверяемое условие истинно. Условие записывается до тела цикла и проверяется до выполнения тела цикла.

Цикл while можно сравнить с условным оператором, блок кода под которым выполняется до тех пор, пока условие не станет ложным.

Оператор break

Оператор break используется для прерывания цикла while.

Оператор continue

Оператор continue начинает следующее повторение, а код следующий после оператора continue не выполняется.

Задача №48

Факториалом числа называется произведение всех чисел от 1 до n.

Например, факториал числа 5 равен 5! = 1*2*3*4*5 = 120, 2! = 1*2 = 2, 9! = 1*2*3*4*5*6*7*8*9 = 362880.
